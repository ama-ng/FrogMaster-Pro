# Eat That Frog Productivity Web App

A modern productivity and anti-procrastination web application inspired by the principles from the book *Eat That Frog!* by Brian Tracy.

This application transforms productivity theories into a practical execution system that helps users:
- Organize tasks
- Prioritize effectively
- Manage goals
- Improve focus
- Build productive habits
- Reduce procrastination
- Increase daily efficiency

Built using:
- HTML5
- CSS3
- Vanilla JavaScript

No frameworks. No backend. No login system.

---

## Features

### Productivity Dashboard
- Daily productivity score
- Focus statistics
- Goal completion tracking
- Frog completion analytics
- Weekly progress overview
- Visual productivity charts

---

### Frog Priority Engine
Advanced task prioritization system based on:
- Biggest frog
- Ugliest frog
- Most impactful task
- Highest consequence task

Features:
- Drag and drop ranking
- Task categorization
- Priority colors
- Completion animations
- Smart sorting system

---

### ABCDE Method System
Tasks are classified using:
- A = Critical
- B = Important
- C = Optional
- D = Delegate
- E = Eliminate

Includes:
- A1, A2, A3 priority ranking
- Smart recommendations
- Daily execution planning
- “Eat First” task section

---

## Offline Persistent Storage

All user data automatically saves using:
- LocalStorage
- IndexedDB

Saved permanently:
- Tasks
- Goals
- Habits
- Timetables
- Analytics
- Focus sessions

Data remains even after:
- Refresh
- Browser close
- Phone shutdown

Only the RESET button clears data.
